# Изменения в заявке проекта

## Функциональные требования

**Заполнение информации** — вместо «о школе»: «об учреждении» с перечислением сущностей (преподаватели, классы, студенты, уроки, аудитории, временные слоты, потоки, учебные группы, привязка уроков к группам, ограничения).  
*Причина:* пользовательский путь стал более удобным.

**Учет аудиторий** — вместо «спортзал только для физкультуры»: «тип, вместимость, оборудование» при размещении занятий.  
*Причина:* реализовать вместимость оказалось проще и актуальнее, чем тип аудитории

**Экспорт** — «в разных форматах (excel, pdf)» заменено на «в PDF».  
*Причина:* реализован только PDF; выгрузка в Excel пока не сделана, продолжаю эксперименты с поиском лучшего формата для экспорта.
